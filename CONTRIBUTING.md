# Contributing to OpenRouter Insights

First off, thank you for considering contributing to **OpenRouter Insights**! It's people like you that make this tool great.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an issue and include as many details as possible, including steps to reproduce.

### Suggesting Enhancements
We are always looking for ways to make the index even better! If you have an idea, please open a feature request issue.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes (`pytest`).
5. Make sure your code lints and is formatted.

## Style Guide
We follow standard Python PEP 8 guidelines. Please keep your code clean and well-commented.
