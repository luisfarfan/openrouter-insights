# Security Policy

## Supported Versions

We currently support the latest stable release of **OpenRouter Insights**.

## Reporting a Vulnerability

If you've discovered a security vulnerability, please report it privately. **Do not create a public issue.**

Please send an email to: [lucho.farfan9@gmail.com](mailto:lucho.farfan9@gmail.com)

We will aim to respond within 48 hours and coordinate a fix.
